WAC-Community-2
===============

A collection of aircraft made for WAC by Dr. Matt and friends.

Installation
===============
1. Put this entire folder into Steam/Steamapps/USERNAME/garrysmod/garrysmod/addons/
2. Done, enjoy your flight.

Aircraft
===============
In this pack you will find:

• MV-22 Osprey
• Bell OH-58D Kiowa
• UH-1D Huey
